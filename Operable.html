Proporcione acceso a todas las funcionalidades mediante el teclado: Los usuarios deben poder navegar por el sitio web y realizar todas las acciones sin necesidad de un ratón.

Conceda a los usuarios tiempo suficiente para leer y usar el contenido: Evite los temporizadores ajustados y proporcione suficiente tiempo para que los usuarios interactúen con el contenido.

No use contenido que pudiera causar convulsiones o reacciones físicas: Evite el uso de animaciones intermitentes o parpadeantes que puedan ser perjudiciales para algunos usuarios.

Ayude a los usuarios a navegar y encontrar el contenido: Utilice una estructura de navegación clara y lógica, títulos y encabezados descriptivos, y un sistema de búsqueda eficaz.

Facilite métodos de entrada diferentes al teclado: Permita que los usuarios interactúen con el sitio web mediante comandos de voz o gestos táctiles.
