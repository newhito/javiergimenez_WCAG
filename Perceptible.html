Proporcione alternativas textuales para contenido no textual: Esto incluye imágenes, gráficos, videos y emojis. Las alternativas textuales deben ser precisas, descriptivas y relevantes para el contenido que representan.

Proporcione subtítulos y otras alternativas para multimedia: Los subtítulos deben ser precisos y sincronizados con el audio, mientras que las transcripciones pueden ser útiles para personas sordas o con dificultades auditivas.

Cree contenido que se pueda presentar de diferentes formas: Esto significa que el contenido debe ser adaptable a diferentes tecnologías de asistencia, como lectores de pantalla o software de ampliación.

Facilite que los usuarios puedan ver y oír el contenido: Use un contraste de color adecuado, evite el texto pequeño y asegure un volumen de audio adecuado.
